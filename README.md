# Tetris implements by go

## Running the Game

Having Go installed, you can run `go run .` from the root directory to play the game.

example:

![](./docs/example1.png)

## Controls

- Left/Right arrow - Move piece
- Up arrow - Rotate piece
- Down arrow - Fast fall
- Space - Instant drop
- Clike - Pause
